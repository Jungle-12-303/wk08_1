# 파트 A — 네트워크 하드웨어 & 커널 송신 경로

> **발표 목적**: write() 한 번이 이더넷 선로까지 나가기까지의 내부 경로를
> 라우팅·재포장(MAC/TTL)·I/O bridge·DMA·sk_buff 레벨까지 그린다.
> **권장 탑다운**: `앱 → user buf → kernel socket buf → TCP/IP 스택 → qdisc → NIC driver → NIC → DMA → PCIe TLP → 선로`
